# 008. Minecraft / Block Common Language Sources

This source note is not a comprehensive bibliography.
The materials listed here are source anchors for checking how the observation target is explained externally, not evidence or foundations for HSS.

These source anchors are not used to prove Minecraft’s educational value or cultural value.
They are also not used to show that Minecraft invented block play, crafting, sandbox design, or world editing.

## Source anchors

### Minecraft Help: All Game Modes in Minecraft

URL:
https://help.minecraft.net/hc/en-us/articles/360058743992

Use:
This is treated as a source anchor for checking that game modes such as Survival, Creative, and Adventure are provided as different connection routes inside Minecraft.
In particular, Creative mode is treated as an auxiliary line for contexts in which the readable unit and writable unit come closer, such as unlimited blocks and instant destruction.

### Minecraft Help: Getting Started with Redstone in Minecraft

URL:
https://help.minecraft.net/hc/en-us/articles/360045950932-Getting-started-with-Redstone-in-Minecraft

Use:
This is treated as a source anchor for checking that Redstone connects to traps, lighting, crafting, automatic farms, automatic doors, and similar forms, and for confirming a context in which the block world reconnects to logic and mechanisms.

### Minecraft Education: GameCode
